# Spotify Music Data Analysis Dashboard (Power BI)

![Spotify Banner](https://github.com/bhanusolanki45/Spotify-PowerBI-Dashboard/blob/main/home.pdf)

## Project Overview
This is an interactive dashboard built in **Microsoft Power BI** to analyze Spotify music data. It provides deep insights into top artists, song popularity, album trends, explicit content distribution, and temporal patterns (monthly/yearly).

The dashboard helps answer questions like:
- Who are the most popular artists and how do their songs perform?
- What are the trends in song popularity over time?
- How does explicit content impact popularity?
- Which albums and singles dominate the charts?

## Key Features
- **Interactive Navigation**: Home, Overview, Artists, and Songs pages.
- **Visualizations**:
  - Gauges and KPIs for total songs, artists, and popularity metrics
  - Bar charts for top artists and songs
  - Line charts for average popularity by month
  - Tables with detailed breakdowns (album type, tracks, max/min popularity)
  - Filters for year, explicit content, and more
- **Insights Highlighted**:
  - Top artists include Taylor Swift, Billie Eilish, The Weeknd, and others
  - Analysis of explicit vs. non-explicit songs
  - Monthly and yearly popularity trends
  - Album vs. single track distribution
